# Восстановление прошивки Meta/Oculus Quest 2/3s/3/Pro

>[!NOTE]
> Если ваш шлем по каким либо причинам, например после обновление прошивки в России (актуально для Quest 2), не загружается в ОС, или после логотипа Meta серый экран **возможно** это руководство поможет вам, в решении этой проблемы.

## Подготовка
> Скачайте архив [adb](https://github.com/MACINTOSH-dev/recovery-quest/releases/download/adb/adb.zip), а затем загрузите **последнюю версию прошивки** для вашей модели (она должна быть последней, это важно!)

#### Загрузка прошивки для вашего шлема
> На [этом сайте](https://cocaine.trade/) выберите свою модель шлема, затем у вас должна открыться страница с прошивками, **нажмите на самую верхнию** в столбике ***Incremental***, затем у вас начнётся загрузка.

#### Подготовка архива
> У вас должен был загрузиться .zip файл (к примеру q3_51312300200800520), **распаковывать его не нужно**, переименуйте его в *Update* (должно получиться так: Update.zip)

## Начало
> Распакуйте где нибудь архив adb, например **С:\adb**, а затем в распакованную папку поместите только что переименовынный файл Update.zip

А затем откройте CMD от имени администратора, и выполните команду. Заменив `путь\к\` на фактический путь к папке adb, которую вы только что распаковали.
```cmd
cd путь\к\adb
```
#### Перевод шлема в режим загрузки

> Полностью выключите шлем, перед этим убедившись что он заряжен не меньше 30%, затем одновременно нажмите и удерживайте кнопки питания и уменьшения громкости, пока не появится экран загрузки.

Затем выберите пункт *Sideload update*, используя кнопки громкости, а затем **нажмите** кнопку питания чтобы подтвердить. Индикатор шлема должен загореться фиолетовым цветом, теперь используя кабель подключите его к ПК. (Используйте кабель с дата линиями, например: кабель link, или кабель который шёл в комплекте со шлемом для зарядки)

### Прошивка
>[!Warning]
> **Все данные со шлема будут стёрты, включая игры, данные об аккаунте, и общие настройки. Я вас предупредил.**

> В окне cmd выполните следующую команду
```cmd
adb.exe  sideload ./Update.zip
```
