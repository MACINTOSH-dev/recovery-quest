# Восстановление прошивки Meta/Oculus Quest 2/3s/3

>[!NOTE]
>Если ваш шлем по каким либо причинам, например после обновление прошивки в России(актуально для Quest 2), не загружается в ОС, или после логотипа Meta серый экран **возможно** это руководство поможет в решении этой проблемы.

### Приступим
>Скачайте архив [adb](https://github.com/MACINTOSH-dev/recovery-quest/releases/download/adb/adb.zip), а затем загрузите **последнюю версию прошивки** для вашей модели (она должна быть последней, это важно!)
